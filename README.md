# SiDeclara — CEDH Sinaloa

Sistema de declaraciones patrimoniales del Órgano Interno de Control
de la Comisión Estatal de Derechos Humanos de Sinaloa.

## Estructura

- `backend/` — API GraphQL (Node.js 14, Express, Apollo, Mongoose)
- `frontend/` — UI (Angular 11, Material, Apollo Angular)

## Producción

Desplegado en VPS 31.220.58.97. Ver `backend/CLAUDE.md` para reglas
operativas y de seguridad.

## Origen

Fork independiente del sistema de PDNMX/SESEA Puebla, adaptado para
CEDH Sinaloa. Sin sincronización activa con upstream.
